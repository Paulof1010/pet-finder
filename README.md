# Pet Finder

A project aimed at helping users find their domestic pets.
This project was part of Code for All_'s Bootcamp #107.

Made by CaniPETS :
Francisco Ramos,
Paulo Ferreira,
Pedro Sequeira e
Rafael Silva.

## Project Purpose
This project intends to provide a platform where users can:
- Browse pets by type (dog, cat, etc.)
- Filter pets by location or breed
- View basic details about lost or sighted pets.

## Tech Stack
This project uses the following technologies:

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Java, Spring Boot  
- **Database:** PostgreSQL 

## Repository Structure
